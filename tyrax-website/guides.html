HAPP + ПОДПИСКА

IPHONE / IPAD

На iOS нативного клиента нет — используем Happ и подписку TYRAX. Протокол: VLESS + Reality + XHTTP.

  1. 01

    Установи Happ

    Скачай из App Store. Happ — клиент для proxy/VLESS, серверы настраиваешь через подписку.

  2. 02

    Получи подписку TYRAX

    Открой Telegram-бот → «iPhone/iPad». Нужен аккаунт TYRAX (войди через приложение на Android/Windows или /start в боте).

  3. 03

    Импортируй URL

    Нажми «Добавить подписку» в боте или Happ → + → Import from URL. Вставь ссылку подписки.

  4. 04

    CONNECT

    Обнови подписку в Happ, выбери сервер TYRAX, подключись. Лимит FREE и дата — в статусной строке Happ.

HAPP + ПОДПИСКА

MAC

Windows — нативный TYRAX. Mac — Happ с той же подпиской VLESS Reality XHTTP.

  1. 01

    Установи Happ

    DMG для macOS (универсальный arm64/intel) или версия из App Store.

  2. 02

    Подписка TYRAX

    Telegram-бот → «Windows/Mac» → «Подписка TYRAX для Mac». Скопируй URL.

  3. 03

    Import from URL

    Happ → + → Import from URL. Подписка обновляется автоматически раз в час.

  4. 04

    CONNECT

    Выбери ноду TYRAX, подключись. Тариф и лимит — в строке статуса Happ и в боте «Мой аккаунт».
